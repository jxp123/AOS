# AOS v1.10 Release Notes — GitHub Update Preflight

## Added
- GitHub Preflight tab.
- `github_settings.json`.
- `Configure_GitHub_Update.bat`.
- Repository URL validation.
- Branch setting.
- Future GitHub ZIP URL generation.
- Update checklist.
- Self-test includes GitHub preflight.
- AI export includes GitHub preflight status.

## Not yet included
- Automatic GitHub download.
- Automatic replacement of files.
- Rollback after failed update.

Those should come after this preflight layer is confirmed stable.
