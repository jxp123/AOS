# AOS v1.7 Release Notes — Natural Language Intake

## Added
- Natural Language Intake tab.
- Rule-based inspection note parser.
- Colony/nuc detection from text.
- Queen, eggs, larvae, brood, stores, bee coverage and temperament extraction.
- Parser confidence score.
- Warnings for weak or ambiguous notes.
- One-click staging into Guided Inspection Drafts.

## Important
The parser is conservative and deterministic. Always review staged drafts before committing.
