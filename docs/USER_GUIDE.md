# AOS User Guide

## First time
Double-click `Install_AOS.bat`.

## Daily use
Double-click `Run_AOS.bat`.

## If broken
Double-click `Repair_AOS.bat`.

## Manual backup
Double-click `Backup_AOS.bat`.

## Important folders
- `data/` contains the live SQLite database.
- `backups/` contains database backups.
- `exports/` contains Excel and AI JSON exports.
- `imports/` is where you place files for import.
- `logs/` contains run logs.
