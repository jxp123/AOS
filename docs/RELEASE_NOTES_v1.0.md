# AOS v1.0 Release Notes

This release consolidates AOS into a stable local foundation.

## Included
- Local SQLite database
- Browser UI via NiceGUI
- Morning briefing
- Central apiary entity view
- Inspections
- Queens
- Equipment
- Genealogy
- Weather / forage
- Seasonal planner
- Validation
- Commit queue
- Excel export
- AI JSON export
- Manual backups

## Known limitations
- No formal migration runner yet
- CRUD is not fully polished for all entities
- Excel import is conservative/scaffolded
- Rules engine is still early-stage
- Real historical reconciliation still required
