# AOS v1.6 Release Notes — Guided Workflow + Staged Commits

## Added
- Guided inspection drafts.
- Validation before staging.
- Evidence preview before staging.
- Confidence score before staging.
- Commit selected draft into inspection log.
- Reject selected draft.
- Tasks now include staged inspection review actions.
- AI export includes guided inspection drafts.

## Why this matters
This release introduces a safer operating model:

Draft → Validate → Stage → Commit

That is the foundation for future natural-language / AI-driven inspection entry.
