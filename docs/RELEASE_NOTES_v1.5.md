# AOS v1.5 Release Notes — AOS 2.0 Foundation

## Added
- Guided Inspection workflow
- Task Engine
- AI Advisor scaffold
- Evidence Engine
- Confidence Engine
- Improved Morning Briefing dashboard
- AI export now includes:
  - knowledge graph
  - generated tasks
  - advisor recommendations
  - evidence/confidence context

## Why this matters
This is the first release where AOS moves beyond records into operating-system behaviour:
- It generates tasks.
- It explains evidence.
- It assigns confidence.
- It guides inspections.
- It starts turning apiary data into decisions.

## Next recommended release
v1.6 should strengthen the guided inspection workflow with stage-by-stage screens, validation gates and automatic staged commits.
