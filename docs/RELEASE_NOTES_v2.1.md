# AOS v2.1 Release Notes — Inspection Scheduler

## Added
- Inspection Scheduler tab.
- Today's Inspection Plan on Morning Briefing.
- Default 7-day inspection rule for active hives and nucs.
- Strategy-based intervals for queen events, recovery/requeening, observation/winter.
- Overdue / due today / due soon / recent statuses.
- 14-day inspection calendar.
- Tasks now include due/overdue inspections.
- AI JSON export includes inspection schedule.
- Self-test includes scheduler check.

## Why
The central question is now explicit: what needs inspecting today?
